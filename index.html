LoliSnatcher Droid 1.5.0

Published on September 24, 2020

I have created an android version of my other project Lolisnatcher, It has been written in Dart with Flutter for the UI. It currently supports more booru engines than the desktop version uncluding:

  • Danbooru
  • Gelbooru
  • Moebooru
  • Philomena
  • Shimmie
  • e621
  • Szurubooru

It should work on all Android devices on versions 4.1.2+. There is also a pre-release Linux build but video playback is currently broken. Booru config files can be moved across from the desktop version they need to be placed in /sdcard/Lolisnatcher/Config

Please find app builds and the source code on Github

LoliSnatcher 1.4

Published on February 10, 2020

I wrote a piece of desktop software in Java for viewing and batch downloading Images from multiple booru websites. It currently supports the Danbooru, Gelbooru and Moebooru engines and is fully themeable with css, custom themes can be loaded from ~/.loliSnatcher/config/theme/active and some example themes are available here

The program has been AOT compiled using Gluon client and does not require a Java runtime. Due to this the executable is quite large but is a lot more memory friendly than a Java build

The source code and binary downloads are avilable on Github

Published on January 09, 2020

I made a Python script which will use ffmpeg to convert anime into a format which is ready for the Playstation portable to play via it's native video player. Softsubs will be burned onto the video track during encoding since the PSP cant use softsubs.
